# 英単語クイズアプリ

![](/src/assets/title.png)

## 概要

英単語のクイズアプリです。

- **難易度変化**
  - スコアに応じて問題の難易度を変化させます。
- **ランキング**
  - 名前とスコアをデータベースに保存しランキングを作成します。

## 技術スタック

- JavaScript
- Vite
- Svelte
- Tailwind CSS
- Supabase
- PostgreSQL
- Vercel

## 環境構築

```sh
npm create vite@latest
npx svelte-add tailwindcss
npm install @supabase/supabase-js
```
